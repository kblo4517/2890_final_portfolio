Rakuma

Previous Next

Description

Rakuma is actually a Japanese company that is a susidiary of Rakuten, a popular e-commerce company similar to Amazon. Rakuma is essentially a place where people can buy and sell secondhand items through a marketplace. Rakuma posted a 99designs contest to commission for the best design for a website for their new phone app. This project was for my summer 2020 portfolio class at Weber State.

Technologies

  • HTML
  • CSS
  • Bootstrap
  • Javascript

Purpose

The website acts as a landing page or informational page where people learn about the Rakuma app, what it can do, and it's features.

Design

I wanted to keep this one simple. The design called for a colorful, simple design so I decided to use the colors from the logo. It provided a good base line for the included colors in the site. Photos that were used included women, since the app is primarily targed towards women in their 30s, along with smartphone usage. The hardest part was finding a screen shot of the application. Turns out the app is only available on the Japanese market so it was hard to obtain. I created my own app screen shot through Photoshop. I also created a second page in Japanese. The opening page asks the user to choose a language and it slightly tailors the design based on that preference.
