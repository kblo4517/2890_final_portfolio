Description

Educate A Mother site is a project I took on from a client. She wanted to redesign the site with an updated look and feel through WordPress. This was my first time working in WordPress, so it was a good learning experience. This site is actually still under construction, but I wanted to include it because it has a clean design. The finished product will be done shortly.

Technologies

  • HTML
  • CSS
  • WordPress

Purpose

Thius site purpose is to connect single mothers through a non-profit organization. The organization works with different resources that mothers can access for additonal help.

Design

Design for this project was straightforward. The entire idea behind the organization is to provide support to single mothers and to point them in the right direction to obtain additional resources if that is needed. One of the definiting characteristics of the site is to help the user become comfortable with using the site and to provide a sense of belonging. For this, I included photos of single mothers such as the hero image for the about page below.

The color scheme that was used was blue and yellow. The reasoning behind this was blue is a nice supporting color that resonates with the cause, whereas using yellow invokes a sense of happiness and strength. In order to keep this theme, the colors were splashed across the site in order to exude a sense of cohesion.
