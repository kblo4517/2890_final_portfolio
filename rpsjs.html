Rock Paper Scissors Javascript

Description

This is a rock, paper, scissors application made with HTML, CSS, and Javascript. The user chooses their option using the buttons at the bottom, while the computer's option is chosen randomnly. The score at the top counts both the user and the computer score infinitely until the user presses quit.

Technologies

  • HTML
  • CSS
  • Javascript

Challenges

This project was actually my first time using Javascript with HTML and CSS. I distinctly remember having issues where the images would continously display in a column. So, if the user pressed rock, the rock picture would show. Then if the user pressed scissors, the scissors picture would show underneath that and so on and so forth. It ended up creating a column of images. Turns out I was adding the images in an additional div instead of just replacing the pre-existing one. This was frustrating to me because it was my first time using JS and I couldn't figure out the issues. Took me a little bit to research and figure it out, but I did and it turned out okay.
