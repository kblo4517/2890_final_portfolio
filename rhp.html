Bootstrap Rent-to-Own Homes Pro

Description

This was actually an assignment completed for my summer 2020 semester at Weber. The idea behind the assignmnet was to redesign a website for a client called Rent-to-Own Homes Pro. The original design was for a contest from 99design and the client expressed interest in a cleaner, more modern design. Below are some of the steps taken to modernize the site.

Technologies

  • HTML
  • CSS
  • Bootstrap
  • SASS

Purpose

The main purpose of this project was to become more comfortable with the Bootstrap framework. Up until this point I only used it sparingly, but never gave it my full attention. This was a chance to really dig deep.

Design/Challenges

The design decisions for this were to design the website so that it was as streamlined as possible. I based the template of the design on a previous project I did with Bootstrap just a few weeks prior, so it made the transition easy since the design was already modernized.

The biggest challenge for this site was getting the photos resized properly to fit the carousel window for the testimonials, and had a few issues with making certain portions of the website responsive. Otherwise, it turned out fine.
