Vue Quotes

Description

This is a quotes application made during my Spring 2020 semester at Weber State University. It lets the user add and delete quotes. At the top is a counter/progress bar that counts to 10. If the user wants to add more quotes, they will need to delete and existing quote by clicking on it.

Technologies

  • Javascript
  • VueJS
  • CSS
  • HTML

Requirements

The requirements for this project were to let the user add quotes using the textbox. Once the progress bar reached 10 quotes, the user would need to delete a quote to add more.
