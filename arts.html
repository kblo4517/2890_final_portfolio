Description

This was my first website redesign. It was for the Ogden First Arts webpage.

Technologies

  • Wix

Purpose

Learning the design process and how to get their by going through, not only the look and feel, but also the content.

Design

There were a lot of slight design changes I made to this that added up to a larger overhaul. Some of the small design changes was adding a description to the Monarch page. This was done because the original page led to some confusion about what the Monarch was. I also added information about locating the venue as well by adding a Gmaps widget on the side and the contact info for the location.

For the events page, I cleaned it up by adding a simple look to the event entries. The title of the event, date, and descriptions in succession. Then with a call to action button if the user wants to learn more about the even in question. On the Panes page, I noticed that the way that the layout for the artists were not optimal for the artists, which is what the page was for. To improve this, I created the layout in a grid format that highlighted the artists more.
